# Lista de Compras

Um exercício simples em Python para gerenciar uma lista de compras com operações básicas de inserção, exclusão e listagem de itens.

## Funcionalidades

- **[i]nserir**: Adiciona um novo item à lista de compras
- **[a]pagar**: Remove um item da lista pelo índice
- **[l]istar**: Exibe todos os itens da lista com seus índices

## Como usar

Execute o script:

```bash
python exercicio_compras.py
```

Selecione uma opção no menu:
- Digite `i` para inserir um item
- Digite `a` para apagar um item
- Digite `l` para listar todos os itens

## Tratamento de Erros

O programa inclui tratamento de erros para:
- Índices inválidos (não numéricos)
- Índices que não existem na lista
- Exceções gerais inesperadas

## Requisitos

- Python 3.x
